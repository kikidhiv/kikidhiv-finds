# KikiDhiv's Finds - Affiliate Shopping Page

A curated lifestyle shopping page featuring handpicked products from Amazon and Meesho. Built with Tailwind CSS and vanilla JavaScript.

## 🚀 Features

- **Dark/Light Mode** - Toggle between themes with persistent preference
- **Platform Filtering** - Filter products by Amazon or Meesho
- **Category Filtering** - Browse by Women, Men, Couple, Kids, and Tech
- **Responsive Design** - Fully mobile-friendly layout
- **Smooth Animations** - Fade-in effects and button shine
